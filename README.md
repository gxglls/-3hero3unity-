# -3hero3unity-
unity版英雄无敌3半成品<br>

  想法的是做一个英雄无敌3+三国群英传2混合玩法的2D策略战棋游戏，战斗系统和经济系统用英3,城市系统用群英2，最后再加入帝国2的兵种克制,想法很美好 <br>

想想英3主要有3大界面 冒险地图界面，战斗界面，城池界面  目前制作了简易的冒险界面和战斗界面 可以移动，捡资源，战斗
